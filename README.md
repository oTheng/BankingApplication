This project is how real life banking would make payments and deposit on their account.

Download the lastest git bash:
https://git-scm.com/install/windows

https://github.com/oTheng/BankingApplication.git
---> git clone https://github.com/oTheng/BankingApplication.git
Download Intellij
https://www.jetbrains.com/idea/download/?section=windows

Open the file clone repo in Intellij
